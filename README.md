# AMLOS React
Production-ready React admin portal with SuperAdmin and School Admin roles.

## Tech Stack
- React 18, Vite, TypeScript 5
- Tailwind CSS 3 + Google Stitch (`@stitches/react`)
- React Router 6, Axios

## Setup
1. `npm install`
2. Create `.env` from `.env.example` and set your `VITE_API_BASE_URL`
3. `npm run dev`
